# NumPy for Data Science

## 📖 Overview
This repository documents my learning journey with NumPy, one of the most important Python libraries for numerical computing and data analysis. It contains concepts, practice exercises, and examples covering the fundamentals of NumPy.

## 🎯 Objectives
- Learn NumPy fundamentals
- Perform efficient numerical computations
- Understand array manipulation techniques
- Explore mathematical and statistical operations
- Build a strong foundation for Data Science and Machine Learning

## 📚 Topics Covered

### 1. NumPy Basics
- Introduction to NumPy
- Creating Arrays
- Array Attributes and Data Types

### 2. Indexing and Slicing
- Accessing Array Elements
- Slicing Arrays
- Boolean Indexing

### 3. Array Operations
- Arithmetic Operations
- Broadcasting
- Array Reshaping
- Concatenation and Splitting

### 4. Mathematical Functions
- Aggregation Functions
- Statistical Functions
- Universal Functions (ufuncs)

### 5. Linear Algebra
- Matrix Operations
- Dot Product
- Matrix Transpose

### 6. Random Module
- Random Number Generation
- Sampling Techniques

## 🛠️ Technologies Used
